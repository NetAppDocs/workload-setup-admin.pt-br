= Gerencie painéis personalizados
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Gerencie seus painéis personalizados a partir do catálogo de painéis. Lá, você pode visualizar detalhes, atualizar dados ou excluir painéis que não precisa mais.

.Sobre esta tarefa
Todos os usuários da sua conta do Workload Factory podem acessar todos os painéis personalizados. Cada painel exibe detalhes básicos, como o nome do criador e a data de criação.

Os painéis personalizados são somente leitura. Eles não podem alterar suas configurações de armazenamento ou carga de trabalho.

.Antes de começar
Você precisa ter pelo menos um painel personalizado em sua conta do Workload Factory.



[[access-the-dashboard-catalog]]
== Acesse o catálogo do dashboard

.Passos
. Faça login usando um dos link:console-experiences.html["experiências de console"^].
. No menu, selecione *Visão geral das cargas de trabalho* e depois *Painéis personalizados*.
+
O catálogo de dashboards mostra todos os dashboards personalizados da sua conta do Workload Factory.

. Revise a lista do dashboard.




[[refresh-dashboard-data]]
== Atualizar dados do dashboard

Os dados do painel são carregados quando você abre um painel. Você pode atualizá-lo para ver as informações mais recentes.

O painel de controle recarrega os dados das suas cargas de trabalho. A hora da última atualização muda para mostrar quando os dados foram atualizados.



[[delete-a-dashboard]]
== Excluir um dashboard

Exclua os painéis que você não precisa mais. Essa ação é permanente e não pode ser desfeita.

.Passos
. Na página *Painéis personalizados*, localize o painel que você deseja remover.
. No menu suspenso *Ações* desse painel, selecione *Excluir*.
. Confirme a exclusão.


.Resultado
O painel de controle é removido permanentemente da sua conta do Workload Factory e todos os usuários perdem o acesso a ele.
